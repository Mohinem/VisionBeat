# Sample Assets

This directory is intentionally committed as **text-only** so pull requests stay compatible with review tooling that does not support binary files.

Generate local development samples with:

```bash
python scripts/generate_demo_samples.py
```

The generated files are:

- `assets/samples/kick.wav`
- `assets/samples/snare.wav`

These WAV files are ignored by Git and should not be committed.
